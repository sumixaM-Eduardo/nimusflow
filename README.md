# nimusflow

NimusFlow is a learning project focused on data engineering fundamentals and system architecture.

The pipeline receives raw sales data, cleans and transforms it, validates business rules and loads the results into a SQLite database. I also built a REST API for future data consumption and containerized the project with Docker to understand how things behave closer to a production environment.

This project covers the basics of the tools I'm currently studying — the goal is to learn by doing and understand how each piece fits together in a real pipeline.
